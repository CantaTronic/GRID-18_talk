We present a current status of data center based on KCDC (KASCADE Cosmic Ray Data Centre), which was originally designed for providing 
an open access to the events measured and analyzed by KASCADE-Grande, a cosmic-ray experiment located in KIT, Karlsruhe. In the frame of
 the Russian-German Astroparticle Data Life Cycle Initiative we extend KCDC in order to provide an access to different cosmic-ray 
experiments and make possible aggregation and joint querying of heterogeneous air-shower data. In the present talk we discuss the 
description of data and metadata structures, implementation of data querying and merging, and first results on including data of 
experiments located in Tunka, Russia, in this common data center. 

Будет прдставлен текущий статус дата-центра, основанного на KCDC, оригинально спроектированного для предоставления открытого доступа к событиям, измеренным и проанализированным на 
установке Каскад-Гранде, эксперименте по изучению комсических лучей из KIT, Карлсруэ. В рамках Россйиско-немецкой инициативы KRAD мы расширили KCDC для предоставления данных 
различных экспериментов по физике космических лучей, возможности агрегации и обработки разнородных даннах по КЛ. В настоящем докладе будет рассмотрено описание данных и метаданных,
организация очередей и способов объединения данных и первые результаты по включению данных эксперимента Tunka (Россия) в данный общий дата-центр.
